def self.join(parts : Array | Tuple)

Returns a new string formed by joining the strings using File::SEPARATOR.

File.join("foo/", "/bar/", "/baz") #=> "foo/bar/baz"
File.join("/foo/", "/bar/", "/baz/") #=> "/foo/bar/baz/"
